# 👥 HR Dashboard - Power BI

## 📌 Proje Hakkında
Bu proje, bir şirketin insan kaynakları verilerini analiz etmek amacıyla 
Power BI ile geliştirilmiş interaktif bir dashboard'dur.

## 📊 Dashboard İçeriği
- **311** toplam çalışan (Headcount)
- **%33,44** işten ayrılma oranı (Attrition)
- **69,02K** ortalama maaş
- **47** ortalama yaş

## 🔍 Analizler
- 📁 Departmana göre Headcount (Production, IT/IS, Sales...)
- 👥 Medeni duruma ve cinsiyete göre dağılım
- 📅 Yıllık Attrition trendi
- 🎯 Yaş gruplarına göre dağılım (Age Bucket)
- 🔗 İşe alım kaynakları analizi (LinkedIn, Indeed, Google...)
- 📈 Kümülatif Headcount büyümesi

## 🛠️ Kullanılan Teknolojiler
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (M Dili)
